# Giderlerim Uygulaması

> NextJS (v.14), Redux ToolKit (v.2), Tailwind kullanılarak geliştirilmiştir.

## Kurulum & Çalıştırma

```
git clone https://github.com/volkansengul/gelir-gider.git

npm i
npm run build
npm start
```
